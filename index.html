FEATURES
  • Complete keyboard accessibility while encouraging speed.
  • One of the most Light weight TO DO APP.
  • Displays time of completion, thus providing a feedback to embrace productivity.
  • Time of the task can be changed if some task left out, but not the content.
  • Tasks will rearrange themselves according to the time
  • With simple UI to controls, one can only be successfull if used honestly.
HOW TO USE
  • + button to create New Task
  • Shift + Enter while inside text area to create New Task
  • Tab or Enter to register Task
  • Up and Down to select Tasks
  • Enter to mark selected Task as Complete
  • Ctrl + Enter to mark selected Task as Incomplete
  • Delete to remove the selected Task
